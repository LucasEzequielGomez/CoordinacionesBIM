BEP

Ejecución BIM es un documento en el que se definen las bases, reglas y normas internas de un proyecto que se va a desarrollar bajo metodología BIM, para que todos los agentes implicados hagan un trabajo coordinado y coherente.

LOD

Level of Detail define el nivel de desarrollo o madurez de información que posee un elemento del modelo, y este es la parte de un componente, sistema constructivo o montaje del edificio. Todos los niveles están determinados por: requerimientos de contenido del elemento, usos autorizados, coste, programación, coordinación u otros.

BIM 4D Y 5D

Desarrollo y coordinación de un Modelo BIM que integre información relacionada con el programa (4D) y costo (5D) del proyecto. El Modelo permite visualizar y simular el proceso de construcción con la finalidad de identificar riesgos y oportunidades para definir la logística de obra. El incluir información de costo dentro del modelo facilita determinar el impacto de cambios en el diseño de una manera ágil y transparente.

Clash Detective

Proceso mediante el cual el Modelo BIM permite la detección oportuna y exhaustiva de omisiones, inconsistencias, conflictos e interferencias entre los elementos y disciplinas que conforman un proyecto. Este proceso permite una mayor coordinación entre los participantes y atender de manera oportuna riesgos y conflictos que pueden representar sobrecostos y retrasos durante la construcción.
